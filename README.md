# Hi there, I'm Jesus Guerra 👋

Senior iOS Engineer with 15+ years of experience designing and evolving production mobile systems across fintech and consumer products. Strong background in modular architecture, secure authentication flows, and performance optimization for large-scale iOS applications.

Based in Hermosillo, Mexico and open to remote roles. Currently collaborating with international teams as an independent iOS consultant, previously contributing to the [Bitso](https://bitso.com/mx) mobile platform.
